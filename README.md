## Descripción
Proyecto desarrollado para la Evaluación Sumativa 4.2 de Desarrollo de Software para Hardware.

El sistema permite recibir datos X-Y desde una aplicación móvil APK y representarlos en una página web utilizando Flask, HTML, CSS y Jinja.

## Tecnologías utilizadas
- Python
- Flask
- HTML
- CSS
- Jinja
- GitHub

## Repositorio base
Se utilizó como base el repositorio entregado por el docente:

https://github.com/jotaefepece/Actividad-dataXY-base

## Importante
No se sube la APK al repositorio personal. La APK solo se utiliza como aplicación externa para enviar datos X-Y.
